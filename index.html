Posicionamiento en CSS

La propiedad position de CSS especifica como un elemento es posicionado en el documento. Las propiedades top, right, bottom, y left determinan la ubicacion final de los elementos posicionados.

Un elemento posicionado es un elemento cuyo valor computado de position es relative , absolute, fixed o sticky. (En otras palabras, cualquiera excepto static).

Usos comunes de posicionamiento en CSS

  • Relative: Se usa para realizar ajustar finos y como base para el posicionamiento absolute.
  • Absolute: Se utiliza para seperposicionamiento de elementos, por ejemplo: cuando tenemos una imagen y sobre ella texto.
  • Fixed: Se utiliza para fijar elementos en la ventana del navegador, como los navbar, algun chatbot, etc.
  • Sticky: Elementos que se pegan en su contenedor al desplazarse, como men칰s laterales

Posicionamiento fixed

El elemento con posicionamiento fijo se posiciona en relacion a ala venta del navegador y permanece fijo cuando se desplaza la pagina.

Posicionamiento sticky

El elemento con posicionamiento sticky se comporta como un elemento relativo hasta que se desplaza a su posicion limite, donde se "pega" y permanece fijo.
